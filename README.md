# Home-Linux-Server

This repository documents the planning, building, and maintenance of my home-built Linux server.
It serves as both a learning project and a long-term reference for system administration tasks.

---

## Goals
- Build a stable Linux-based home server
- Practice real-world system administration
- Monitor system health and services
- Learn networking, storage, and automation
- Document everything clearly for future reference

---

## Current Status
- OS installed (Linux)
- Core services running (SSH, cron, NetworkManager)
- System monitoring script in progress (`sysinfo.sh`)
- Disk inventory and recovery verified
- Network diagnostics implemented
